# LandInfo – Digital Land Records Information System

## Overview
LandInfo is a web-based platform developed to demonstrate how land-related information can be organized and accessed digitally. The system provides structured access to important land services such as **Record of Rights (ROR), Encumbrance Certificate (EC), prohibited land details, and land document information**.

The objective of this project is to simulate a digital land record system where users can easily navigate through different services and view land-related information through a simple and user-friendly interface.

This project was developed as part of an **academic learning initiative** to understand web development concepts, website structure, and digital public service platforms.

---

## Objectives
- To develop a simple web platform for accessing land information
- To demonstrate the structure of digital land record systems
- To create an easy-to-use interface for land-related services
- To understand basic web development using HTML, CSS, and JavaScript
- To organize land information services in a structured format

---

## Features
- Simple and user-friendly interface
- Structured land information pages
- Record of Rights (ROR) information section
- Encumbrance Certificate (EC) details page
- Prohibited land information page
- Land documents information section
- Admin login interface

## Project Significance
Digital land information systems help improve **transparency, accessibility, and efficiency** in managing land records. This project demonstrates how land-related services can be structured into a digital platform where users can easily explore and access different information pages.
